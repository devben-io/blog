## Ubuntu {#ubuntu}

```
cat /etc/lsb-release
DISTRIB_ID=Ubuntu
DISTRIB_RELEASE=9.04
DISTRIB_CODENAME=jaunty
DISTRIB_DESCRIPTION="Ubuntu 9.04"
```
<!--more-->
## Debian {#debian}

```
cat /etc/debian_version
5.0.2
```

## CentOS {#centos}

```
cat /etc/centos-release
CentOS Linux release 7.0.1406 (Core)
```

## RedHat {#redhat}

